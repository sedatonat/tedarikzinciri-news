# Tedarik Zinciri Haberleri

Tedarik zinciri, lojistik, satınalma, envanter ve teknoloji alanındaki güncel haberlerin Türkçe ve İngilizce paylaşıldığı haber portalı.

**Canlı:** [tzp.news](https://tzp.news)

## Teknoloji Yığını

- [Astro](https://astro.build) — statik site üreteci
- [Tailwind CSS](https://tailwindcss.com) — stil sistemi
- [Cloudflare Pages](https://pages.cloudflare.com) — hosting
- Türkçe/İngilizce çift dil (Astro i18n)
- Light/dark tema

## Geliştirme

```bash
npm install
npm run dev
```

## Yapı

```
src/
  content/news/   # haber Markdown dosyaları
  components/     # Astro bileşenleri
  layouts/        # sayfa şablonları
  pages/          # URL rotaları
  utils/          # yardımcı fonksiyonlar
public/
  images/news/    # haber görselleri
scripts/          # migrasyon/otomasyon scriptleri
```
